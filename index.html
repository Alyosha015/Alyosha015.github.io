StormTools

Using my questionable JavaScript ability and inspiration from MrLennyn's Website, I've made this to have versions of my stormworks tools available to use, since releasing a bunch of exe's seemed excessive.

For now I've only added my image to paintblock converter and vehicle weatherer, but could add more in the future.
For suggestions, reporting bugs, or questions, either submit an issue on Github or on discord: Alyosha15#8170.
