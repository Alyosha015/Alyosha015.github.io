Vehicle Weatherer

In Stormworks, weathering generally is when faces are painted with random slight variations in the colors to make them appear weathered. With this tool you pick a specfic color on the vehicle to weather, and the amount of colors to weather with.

Vehicle Weatherer v0.1.2




Tutorial

  1. Click Select File and select file of the vehicle you want to weather. Vehicles files are stored in %AppData%\Stormworks\data\vehicles\.
  2. Enter the rgb values of the color on your vehicle you want to be weathered in the Color To Weather text box. They have to be entered with spaces inbetween, for example 234 124 86.
  3. Set the variation of the colors used to weather from the original color by changing Color Var.. This controls the max amount the rgb values of the colors used to weather will differ from the colors being weathered. If you want to set your own colors with the color replace tool in the vehicle editor, I suggest setting the value to 255 to see the difference better. If not I suggest keeping it in the 2-20 range.
  4. Set how many colors should be used to weather by changing the # of Colors number input. (This includes the original color, for example if set it to 4, 3 extra colors would be used).
  5. If you want the amount of blocks to be weathere to be lower, decrease the % to Weather number input. It controls what percentage of the faces have their color modified.
  6. Click Generate XML, and the vehicle file will be generated and downloaded. Copy if from your downloads folder to the stormworks vehicle folder: %AppData%\Stormworks\data\vehicles\. (One way to reach the folder is to press Windows+R to open Run, then paste the path in and press enter).
